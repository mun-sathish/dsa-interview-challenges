### [Jump Game II](https://leetcode.com/problems/jump-game-ii/)

- Use greedy approach to find minimum jumps
- Track current jump end and farthest reachable
- Increment jumps when reaching current end
- Update current end to farthest reachable

```js copy
/**
 * @param {number[]} nums
 * @return {number}
 */
var jump = function(nums) {
    let jumps = 0;
    let currentEnd = 0;
    let farthest = 0;
    
    for (let i = 0; i < nums.length - 1; i++) {
        farthest = Math.max(farthest, i + nums[i]);
        
        if (i === currentEnd) {
            jumps++;
            currentEnd = farthest;
            
            if (currentEnd >= nums.length - 1) {
                break;
            }
        }
    }
    
    return jumps;
};
```

##### Complexity

- Time: O(n) where n is length of array
- Space: O(1)

### [Group Anagrams](https://leetcode.com/problems/group-anagrams/)

- Use hash map with sorted string as key
- Group strings with same sorted form together
- Return all groups as result

```js copy
/**
 * @param {string[]} strs
 * @return {string[][]}
 */
var groupAnagrams = function(strs) {
    const map = new Map();
    
    for (let str of strs) {
        const sorted = str.split('').sort().join('');
        
        if (!map.has(sorted)) {
            map.set(sorted, []);
        }
        
        map.get(sorted).push(str);
    }
    
    return Array.from(map.values());
};
```

##### Complexity

- Time: O(n * k log k) where n is number of strings and k is max length
- Space: O(n * k) for hash map

### [Climbing Stairs](https://leetcode.com/problems/climbing-stairs/)

- Use dynamic programming (Fibonacci pattern)
- At each step, can come from previous or two steps back
- dp[i] = dp[i-1] + dp[i-2]
- Optimize space by using two variables

```js copy
/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
    if (n <= 2) return n;
    
    let prev2 = 1;
    let prev1 = 2;
    
    for (let i = 3; i <= n; i++) {
        const current = prev1 + prev2;
        prev2 = prev1;
        prev1 = current;
    }
    
    return prev1;
};
```

##### Complexity

- Time: O(n) where n is number of steps
- Space: O(1)

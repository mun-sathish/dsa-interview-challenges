### [Insert Interval](https://leetcode.com/problems/insert-interval/)

- Add all intervals before new interval
- Merge all overlapping intervals with new interval
- Add all intervals after new interval
- No need to sort as input is already sorted

#### Visual Explanation

**Three-Phase Insertion:**
```
intervals = [[1,3],[6,9]], newInterval = [2,5]

Timeline:
|---|           [1,3]
  |---|         [2,5] (new)
      |---|     [6,9]

Phase 1: Add intervals BEFORE new
  None (1 overlaps with 2)

Phase 2: MERGE overlapping
  [1,3] overlaps [2,5]
  Merge: [min(1,2), max(3,5)] = [1,5]

Phase 3: Add intervals AFTER new
  [6,9] (no overlap with [1,5])

Result: [[1,5],[6,9]]
```

**Overlap check:**
```

```

```js copy
/**
 * @param {number[][]} intervals
 * @param {number[]} newInterval
 * @return {number[][]}
 */
var insert = function(intervals, newInterval) {
    const result = [];
    let i = 0;
    
    // Add all intervals that come before newInterval
    while (i < intervals.length && intervals[i][1] < newInterval[0]) {
        result.push(intervals[i]);
        i++;
    }
    
    // Merge all overlapping intervals with newInterval
    while (i < intervals.length && intervals[i][0] <= newInterval[1]) {
        newInterval[0] = Math.min(newInterval[0], intervals[i][0]);
        newInterval[1] = Math.max(newInterval[1], intervals[i][1]);
        i++;
    }
    result.push(newInterval);
    
    // Add all intervals that come after newInterval
    while (i < intervals.length) {
        result.push(intervals[i]);
        i++;
    }
    
    return result;
};
```

##### Complexity

- Time: O(n) where n is number of intervals
- Space: O(n) for output array

### [Maximum Subarray](https://leetcode.com/problems/maximum-subarray/)

- Use Kadane's algorithm for optimal solution
- Track current sum and maximum sum seen
- Reset current sum if it becomes negative
- Update maximum at each step

#### Visual Explanation

**Kadane's Algorithm:**
```
nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]

i=0: num=-2, current=-2, max=-2
i=1: num=1, current=1 (reset), max=1
i=2: num=-3, current=-2, max=1
i=3: num=4, current=4 (reset), max=4
i=4: num=-1, current=3, max=4
i=5: num=2, current=5, max=5
i=6: num=1, current=6, max=6 âœ“
i=7: num=-5, current=1, max=6
i=8: num=4, current=5, max=6

Best subarray: [4,-1,2,1] = 6
```

**Key insight:**
```
If current < 0: Start fresh
Else: Keep adding to current
```

```js copy
/**
 * @param {number[]} nums
 * @return {number}
 */
var maxSubArray = function(nums) {
    let maxSum = nums[0];
    let currentSum = nums[0];
    
    for (let i = 1; i < nums.length; i++) {
        currentSum = Math.max(nums[i], currentSum + nums[i]);
        maxSum = Math.max(maxSum, currentSum);
    }
    
    return maxSum;
};
```

##### Complexity

- Time: O(n) where n is length of array
- Space: O(1)

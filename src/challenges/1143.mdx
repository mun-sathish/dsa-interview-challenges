### [Longest Common Subsequence](https://leetcode.com/problems/longest-common-subsequence/)

- **DP Pattern**: 2D DP for string matching
- dp[i][j] = LCS length of text1[0...i-1] and text2[0...j-1]
- If characters match: dp[i][j] = dp[i-1][j-1] + 1
- Else: dp[i][j] = max(dp[i-1][j], dp[i][j-1])

- (a,c): no match â†’ dp[1][2] = max(dp[0][2], dp[1][1]) = 1
- (c,c): match â†’ dp[3][2] = dp[2][1] + 1 = 2

**Key insight:**
```
If text1[i-1] == text2[j-1]:
    dp[i][j] = dp[i-1][j-1] + 1  (extend LCS)
Else:
    dp[i][j] = max(dp[i-1][j], dp[i][j-1])  (skip one char)
```

```js copy
/**
 * @param {string} text1
 * @param {string} text2
 * @return {number}
 */
var longestCommonSubsequence = function(text1, text2) {
    const m = text1.length;
    const n = text2.length;
    
    // dp[i][j] = LCS length of text1[0..i-1] and text2[0..j-1]
    const dp = Array.from({ length: m + 1 }, () => Array(n + 1).fill(0));
    
    for (let i = 1; i <= m; i++) {
        for (let j = 1; j <= n; j++) {
            if (text1[i - 1] === text2[j - 1]) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }
    
    return dp[m][n];
};
```

##### Complexity

- Time: O(m * n) where m and n are string lengths
- Space: O(m * n) for dp table

### [Jump Game](https://leetcode.com/problems/jump-game/)

- Track the farthest position reachable
- For each position, check if it's reachable
- Update farthest position based on current jump
- Return true if we can reach the last index

#### Visual Explanation

**Greedy Reachability:**

        0  1  2  3  4

     Can reach: [0,1,2]

i=1: maxReach = 1 + 3 = 4

        0  1  2  3  4

```js copy
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var canJump = function(nums) {
    let maxReach = 0;
    
    for (let i = 0; i < nums.length; i++) {
        if (i > maxReach) {
            return false;
        }
        
        maxReach = Math.max(maxReach, i + nums[i]);
        
        if (maxReach >= nums.length - 1) {
            return true;
        }
    }
    
    return true;
};
```

##### Complexity

- Time: O(n) where n is length of array
- Space: O(1)

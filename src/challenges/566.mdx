### [Reshape the Matrix](https://leetcode.com/problems/reshape-the-matrix/)

- Check if reshape is possible: mÃ—n must equal rÃ—c
- Flatten matrix to 1D, then reshape to new dimensions
- Use formula: row = index / c, col = index % c
- Single pass through original matrix

```js copy
/**
 * @param {number[][]} mat
 * @param {number} r
 * @param {number} c
 * @return {number[][]}
 */
var matrixReshape = function(mat, r, c) {
    const m = mat.length;
    const n = mat[0].length;
    
    // Check if reshape is possible
    if (m * n !== r * c) {
        return mat;
    }
    
    const result = Array.from({ length: r }, () => Array(c).fill(0));
    
    for (let i = 0; i < m * n; i++) {
        // Original matrix position
        const origRow = Math.floor(i / n);
        const origCol = i % n;
        
        // New matrix position
        const newRow = Math.floor(i / c);
        const newCol = i % c;
        
        result[newRow][newCol] = mat[origRow][origCol];
    }
    
    return result;
};
```

##### Complexity

- Time: O(m Ã— n) single pass through all elements
- Space: O(m Ã— n) for result matrix

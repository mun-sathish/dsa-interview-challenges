### [Toeplitz Matrix](https://leetcode.com/problems/toeplitz-matrix/)

- Check if every diagonal from top-left to bottom-right has same elements
- For each cell, compare with cell at (i-1, j-1)
- Can optimize by checking only necessary cells
- Simple one-pass solution

#### Visual Explanation

```js copy
/**
 * @param {number[][]} matrix
 * @return {boolean}
 */
var isToeplitzMatrix = function(matrix) {
    const m = matrix.length;
    const n = matrix[0].length;
    
    for (let i = 1; i < m; i++) {
        for (let j = 1; j < n; j++) {
            // Check if current cell equals top-left diagonal neighbor
            if (matrix[i][j] !== matrix[i - 1][j - 1]) {
                return false;
            }
        }
    }
    
    return true;
};
```

##### Complexity

- Time: O(m √ó n) check each cell once
- Space: O(1)

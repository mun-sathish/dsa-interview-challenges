### [Meeting Rooms](https://leetcode.com/problems/meeting-rooms/)

- Sort intervals by start time
- Check if any two consecutive intervals overlap
- If current start < previous end, there's overlap
- Return false if overlap found, true otherwise

```js copy
/**
 * @param {number[][]} intervals
 * @return {boolean}
 */
var canAttendMeetings = function(intervals) {
    if (intervals.length <= 1) return true;
    
    // Sort by start time
    intervals.sort((a, b) => a[0] - b[0]);
    
    for (let i = 1; i < intervals.length; i++) {
        // Check if current meeting starts before previous ends
        if (intervals[i][0] < intervals[i - 1][1]) {
            return false;
        }
    }
    
    return true;
};
```

##### Complexity

- Time: O(n log n) for sorting
- Space: O(1) excluding sort space

### [Burst Balloons](https://leetcode.com/problems/burst-balloons/)

- **Advanced DP Pattern**: Interval DP
- Think backwards: which balloon to burst last in a range
- dp[i][j] = max coins from bursting balloons between i and j
- Add virtual balloons with value 1 at boundaries

#### Visual Explanation

**Interval DP (nums=[3,1,5,8], balloons=[1,3,1,5,8,1]):**
```
Think: Which balloon to burst LAST in each range?

Range [1,3,1,5,8,1]:
- Burst 3 last: 1*3*1 + dp[1][3] + dp[3][8,1]
- Burst 1 last: 1*1*1 + dp[1][1] + dp[1][8,1]
- Burst 5 last: 1*5*1 + dp[1][5] + dp[5][8,1]
- Burst 8 last: 1*8*1 + dp[1][8] + dp[8][1]

Build from smaller ranges:
len=2: [1,3,1], [3,1,5], [1,5,8], [5,8,1]
len=3: [1,3,1,5], [3,1,5,8], [1,5,8,1]
len=4: [1,3,1,5,8], [3,1,5,8,1]
len=5: [1,3,1,5,8,1]

Example: Range [1,3,1,5]
- Burst 3 last: 1*3*5 = 15 + dp[1][3] + dp[3][5]
- Burst 1 last: 1*1*5 = 5 + dp[1][1] + dp[1][5]
Choose max!

Result: 167 coins
```

**Key insight:**
```
Think backwards: Last balloon burst has fixed neighbors
dp[left][right] = max over k of:
    balloons[left] * balloons[k] * balloons[right]
    + dp[left][k] + dp[k][right]
```

```js copy
/**
 * @param {number[]} nums
 * @return {number}
 */
var maxCoins = function(nums) {
    // Add virtual balloons with value 1 at boundaries
    const balloons = [1, ...nums, 1];
    const n = balloons.length;
    
    // dp[i][j] = max coins from bursting balloons between i and j (exclusive)
    const dp = Array.from({ length: n }, () => Array(n).fill(0));
    
    // len is the length of the range
    for (let len = 2; len < n; len++) {
        for (let left = 0; left < n - len; left++) {
            const right = left + len;
            
            // Try bursting each balloon k last in range (left, right)
            for (let k = left + 1; k < right; k++) {
                const coins = balloons[left] * balloons[k] * balloons[right];
                const total = coins + dp[left][k] + dp[k][right];
                dp[left][right] = Math.max(dp[left][right], total);
            }
        }
    }
    
    return dp[0][n - 1];
};
```
